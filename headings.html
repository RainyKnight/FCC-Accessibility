Headings!

Headings range from <h1>...<h6>

Accessibility rules for headings:

  • There should be only one <h1> element on a page
  • Headings can be used to organize information into hierarchies
    • For example a <h4> should be a detail of a <h3>

To quote FreeCodeCamp:

"Headings with equal (or higher) rank start new implied sections, headings with lower rank start subsections of the previous one."
