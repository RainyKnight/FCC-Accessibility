Pictures with alt text!

Alt text is required on every picture on a website following current accessibility guidelines. This helps blind people using screen readers to understand what the pictures represent.

Layout

Headings!

Headings range from <h1>...<h6>

Accessibility rules for headings:

  • There should be only one <h1> element on a page
  • Headings can be used to organize information into hierarchies
    • For example a <h4> should be a detail of a <h3>

This is because most screen readers tab through information by heading type.

To quote FreeCodeCamp:

"Headings with equal (or higher) rank start new implied sections, headings with lower rank start subsections of the previous one."
Here's an example of hierarchical layout:
  • <h1> Cool Facts About Penguins
    • <h2> Favorite food
      • <h3> Fish
        • <h4> Albacore tuna
      • <h3> Plants
        • <h4> Kelp
    • <h2> - Favorite activities
      • <h3> Swimming
      • <h3> Sleeping
      • <h3> Staring at the sky longingly
      • <h3> Huddling

Nav

Each page should have a nav element at the top to allow the reader to easily navigate between pages.

Header

The header element was derived from the <head> element, but now primarily surrounds heading elements to help organize content.

Color

High Contrast Colors

Colors are an important part of a web page. Colorblind users should be taken into account when deciding on a color palette.

It's important to take the contrast ratio into consideration when deciding on a color.

Some designers even recommend building the website in grayscale first then adding color throughout the site to accommodate folks with monochromacy. This ensures that the luminescence of each element is sufficiently different enough.

According to FreeCodeCamp:

"The Web Content Accessibility Guidelines (WCAG) recommend at least a 4.5 to 1 contrast ratio for normal text. The ratio is calculated by comparing the relative luminance values of two colors. This ranges from 1:1 for the same color, or no contrast, to 21:1 for white against black, the strongest contrast. There are many contrast checking tools available online that calculate this ratio for you."

Check out this contrast ratio checker to determine if your website meets the 4.5:1 standard.

Links

People often just set the words "click here" as the link to a page. Many screen readers will only read out what's between the anchor tags of a link, ignoring nearby context. As a result, it's helpful to put text describing the link between the anchor tags.

With the element, the creator can completely customize the order in which parts of a page are tabbed through.
